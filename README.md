# UniHub Platform 🚀

UniHub is a premium, unified campus portal designed to streamline student life. It connects a modern React web frontend with a fast Express API backend and a cloud-hosted Supabase PostgreSQL database.

---

## 🌟 Features & Modules
1. **Academics Hub**: Share and trade textbooks with peers, download digital syllabus books, and upload lecture notes/manuals (supports direct PDF uploads to the server).
2. **Lost & Found Bulletin**: Report lost objects on campus, validate ownership details, and submit claims in a modern, premium dark interface.
3. **Smart Canteen**: Skip queues by ordering food directly from the classroom and tracking preparation live.
4. **Venue Coordinator**: Request reservations for seminar halls, classrooms, and labs for campus events.

---

## 🛠️ Tech Stack
* **Frontend**: React.js, Vite, Tailwind CSS, Lucide Icons, Framer Motion
* **Backend**: Node.js, Express.js, Multer (file uploads)
* **Database**: Supabase PostgreSQL (Cloud Database)

---

## 🚀 How to Run Locally

### 1. Run the Express Backend API
```bash
cd backend
npm install
node index.js
```
*The server will run on `http://localhost:4000`.*

### 2. Run the Vite React Frontend
Open a new terminal tab and run:
```bash
cd frontend
npm install
npm run dev
```
*The application will open on `http://localhost:5173` (or `5174`).*
